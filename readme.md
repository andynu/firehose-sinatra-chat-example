simple chat server using firehose.io

# PRE-REQS

  * Ruby

  http://www.ruby-lang.org/en/
  I like rvm ( https://rvm.io/ ).

  * bundler gem

    $ gem install bundler

  * Redis

  Ubuntu

    $ sudo apt-get install redis-server

  OSX

    $ brew install redis

# DEPENDENCIES

    $ bundle installing

# RUNNING

    $ foreman start
